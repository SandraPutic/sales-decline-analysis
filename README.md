# Sale Decline Analysis

## Описание проекта
В компании наблюдается снижение выручки. Цель проекта — проанализировать данные о продажах, выявить причины снижения и предложить рекомендации для улучшения показателей.

##  Задачи
- Проанализировать динамику выручки
- Оценить количество заказов
- Изучить поведение клиентов
- Рассчитать средний чек
- Определить ключевые факторы снижения выручки

##  Инструменты
- SQL

##  Анализ

### Выручка
В марте наблюдается снижение выручки по сравнению с предыдущими месяцами.

### Количество заказов
Количество заказов снизилось, что указывает на уменьшение активности клиентов.

### Клиенты
Число уникальных клиентов уменьшилось.

### Средний чек
Средний чек также снизился, что дополнительно повлияло на падение выручки.

##  Выводы
Основная причина снижения выручки — уменьшение активности клиентов, а также снижение среднего чека.

##  Рекомендации
- Внедрить механики удержания (email-рассылки, акции)
- Стимулировать повторные покупки
- Провести сегментацию клиентов для более точечного маркетинга

##  Ключевой инсайт
Снижение выручки связано как с уменьшением количества клиентов, так и со снижением их среднего чека.
